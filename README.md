# 📚 DAA Study Tracker  
### Design & Analysis of Algorithms – Smart Study Planner  

This is a personal project built by me to **track my own progress for my college Semester 4 DAA syllabus** in a structured and consistent way.  

Instead of random studying, this tracker helps me stay organized, practice regularly, and complete the syllabus with clarity and discipline.  

---

## 🚀 Features  

### 📖 Complete Syllabus Coverage  
- Unit 1: Asymptotic Analysis & Searching  
- Unit 2: Sorting & Divide & Conquer  
- Unit 3: Graph Algorithms  
- Unit 4: Greedy & Dynamic Programming  
- Unit 5: Hashing, String Matching & NP Problems

---

### 📊 Progress Tracking  
- Topic-wise completion tracking  
- LeetCode problem tracking  
- Overall progress percentage  
- Daily study streak system  

---

### 🧠 Smart Practice  
- Curated LeetCode problems for each topic  
- Difficulty levels (Easy / Medium / Hard)  
- Helps in both theory + coding preparation  

---

### 📝 Notes & Customization  
- Add personal notes for each topic  
- Add your own topics  
- Add custom LeetCode questions  

---

### ⏳ Exam Focus  
- Countdown to exam  
- Keeps preparation aligned with time  

---

### 🧪 Mock Tests  
- Unit-wise mock tests  
- 3-hour exam simulation  
- Practice like real exams  

---

## 🖥️ Tech Stack  

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- LocalStorage (to save progress)
