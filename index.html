🧠 Key Exam Tips
GREEDY ✓ WHEN
• Locally optimal = globally optimal
• Activity selection, Fractional knapsack
• Huffman coding, MST
• Interval scheduling problems
DP ✓ WHEN
• Overlapping subproblems exist
• Optimal substructure holds
• 0/1 Knapsack, LCS, LIS
• Count ways / min cost problems
GRAPH: BFS vs DFS
• BFS → shortest path (unweighted)
• DFS → cycle detection, topo sort
• BFS → level-order, multi-source
• DFS → connected components
NP KEY POINTS
• P ⊆ NP (unproven P ≠ NP)
• NP-Complete: hardest in NP
• Verify in poly time ≠ Solve in poly
• Reduction: A ≤_p B
